Flyhub Airline reservation systems (FARS) is a system that allow an airline to sell their inventory (seats). It contains information on schedules and fares and contains a database of reservations (or passenger name records) and of tickets issued (if applicable). ARSs are part of Passenger Service System (PSS), which are applications supporting the direct contact with the passenger.

FARS eventually evolved into the Computer Reservations System (CRS). A computer reservation system is used for the reservations of a particular airline and interfaces with a Global Distribution System (GDS) which supports travel agencies and other distribution channels in making reservations for most major airlines in a single system.
